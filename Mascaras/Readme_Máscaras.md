# Generar Máscaras 
_Para la generación de las máscaras de los ROIs en las diferentes secuencias de un estudio, es necesario tene para cada estudio una carpeta con las secuencias almacenadas en .nrrd, como se puede observar en la carpeta **Estudios**. 

La ejecución del código debe ser realizada desde **main_mascaras.py**, en el cuál es usado un archivo **Data_3_slices**, que contiene en cada fila los datos necesarios de cada ROI para generar la máscara (Estudio al que pertenece)
